üìö Unit 1 Vocabulary

  • Counter-culture

  • Authenticity

    An artistic technique of assembling different materials (paper, photographs, fabric) onto a surface to create a new composition
  • Value
    The lightness or darkness of colors and tones, used to create depth, emphasis, and dramatic effect
  • Contrast
    The difference between elements (light/dark, rough/smooth, large/small) used to create visual interest and emphasis
  • Composition
    The arrangement of visual elements within an artwork to create balance, emphasis, and meaning
  • Mixed Media
    Artwork that combines multiple materials and techniques in a single piece
